# 📖 Sobre

Este repositório reúne todos os meus estudos sobre os **fundamentos da linguagem Java**.

O objetivo é documentar minha evolução como desenvolvedor por meio de exemplos práticos, exercícios e implementações realizadas durante o curso **DevDojo** e estudos complementares.

Além de servir como material de consulta, este repositório faz parte do meu **portfólio no GitHub**, permitindo acompanhar minha evolução ao longo do tempo através do histórico de commits.

---

# 🎯 Objetivos

- Aprender os fundamentos da linguagem Java
- Consolidar conceitos através da prática
- Registrar minha evolução como desenvolvedor
- Manter um histórico organizado dos estudos
- Construir um portfólio técnico consistente
